# Привычки — трекер привычек для Android

Минималистичное нативное Android-приложение для отслеживания ежедневных
привычек. Без регистрации, работает офлайн, все данные хранятся на устройстве.

Написано на Kotlin + Jetpack Compose.

## Возможности

- Отметка привычек в один тап с подсчётом стриков
- Недельная сетка прогресса в каждой карточке
- Прогресс дня в процентах
- 12 иконок и 6 цветов на выбор
- Ежедневные напоминания
- Статистика: метрики и график за неделю
- Тёмная тема (системная / светлая / тёмная)
- Локальное хранение (Room), без интернета и аккаунтов

## Технологии

- Kotlin
- Jetpack Compose (Material 3)
- Room — локальная база данных
- DataStore — настройки
- AlarmManager + Notifications — напоминания
- Navigation Compose — навигация

## Сборка

1. Открой проект в Android Studio (File → Open).
2. Дождись Gradle Sync.
3. Нажми Run или собери APK: Build → Build APK(s).

Требуется Android Studio (Koala+) и устройство/эмулятор с Android 7.0 (API 24)
или новее. Целевой и компилируемый SDK — Android 15 (API 35).
